# 桥梁

工程互联网的三个项目之一。


## 任务目标

### 任务一：预测桥梁挠度

现场随机组合车辆荷载并称重各组，使用自建算法预测桥梁挠度，比较预测值和激光测距值差异，重复三次取最高分。

说明：激光位移计会随机放在图 1 所示的红色范围内。已知小车质量后，让小车绕模型桥行驶一周，或放置在多个不同位置，通过加速度数据、小车位置数据、应变数据等分析，求出激光位移计安装位置处的最大挠度，并与激光位移计实测结果对比。每次工况中激光位移计位置和小车质量都可能变化。

### 任务二：预测车辆质量

现场随机组合车辆荷载并加载各组，使用自建算法预测车辆质量，比较预测质量与称重值差异，重复三次取最高分。

说明：小车质量随机改变，激光位移计固定在图 1 所示的红色范围内。让小车绕模型桥行驶一周，或放置在多个不同位置，通过加速度数据、激光位移计数据、小车位置数据、应变数据等分析，反推出小车质量，并与实际称重结果对比。每次工况中小车质量都会变化。

### 任务三：估计结构基频

使用自建算法计算结构基频，比较计算值与设计值差异。可给出三个估值，取最高分。

说明：从之前课程中测得的数据中选取三段进行分析，得到模型桥基频，并在 PPT 中对比三段数据计算出的基频结果和设计值之间的差异，同时说明求解思路和所用算法。

## 附加任务

- 附加任务一：通过手机录制视频，建立点云。
- 附加任务二：根据视频，预测位移。

## 文件

- `data/`：原始数据，包含加速度、应变、位移、小车轨迹等信息。
- `annotation/`：标注数据，包含每个数据点的时间标签。
- `sensor_def.json`：传感器定义文件，描述了每个传感器的类型、位置和测量内容。
- `*.html`：数据可视化网页，用于查看传感器分布和数据展示。
- `submit/`：最终上交的文件。

## 脚本

- `train.py`：训练多信号连续模型。
- `predict.py`：使用训练好的模型进行预测，可通过 `--sensor-cm`、`--load-cm`、`--disp-rel`、`--mass` 指定输入。
- `submit/convert_long_to_wide.py`：把竖表数据转换成横表数据。
- `submit/plot_*.py`：绘制传感器和试验结果图像，辅助数据分析。

训练后会在 `outputs/` 下生成结果文件，例如参数表、散点图、曲面拟合图、三维拟合图和模型 JSON。
